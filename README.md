# size
Размер папок php
Просто загрузи size.php в корень сайта и запусти его из браузера сайт.ру/size.php.

Кликнув на папку ее можно открыть и посмотреть более детально 
Серый - байты 
Зеленый - килобайты 
Оранжевый - мегабайты 
Красный - гигобайты
